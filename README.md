# File explorer
### *Quick* setup guide
**Requirements:**
```
Windows only (Linux and Mac in future)
Rust - Installed
Tauri-cli - Installed
npm - INstalled
```
**Install all dependencies:**
```shell
npm install
```
**Run on development:**
```shell
npm run tauri dev
```
**Build into installer:**
```shell
npm run tauri build
```
**Installer:**
> /src-tauri/target/release/bundle
>
> /msi - a msi Installer
> 
> /nsis - an exe Installer
